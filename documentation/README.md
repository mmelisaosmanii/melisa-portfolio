# AI Assistant for NGOs 

## Overview

AI Assistant for NGOs is an intelligent digital platform designed to help non-governmental organizations improve their daily workflow using artificial intelligence.

The system helps NGO staff manage documents, generate reports, translate content, and communicate with an AI-powered assistant.

---

## Problem

Many NGOs spend significant time managing documents, preparing reports, and processing information manually.

This project aims to reduce repetitive tasks and improve productivity by providing AI-powered automation tools.

---

## Main Features

- AI chatbot assistant
- Document summarization
- Document translation
- AI-generated reports
- User authentication
- Role-based access control
- Document management
- Dashboard overview

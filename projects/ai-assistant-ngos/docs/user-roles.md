# User Roles and Permissions

## 1. NGO Staff

### Description
Regular users who work inside an NGO and use the AI assistant for daily tasks.

### Permissions
- Create an account
- Login to the system
- Upload documents
- Use AI chatbot
- Generate reports
- View personal documents
- View personal conversations


---

## 2. Project Manager

### Description
Users responsible for managing NGO projects and reviewing information.

### Permissions
- All NGO Staff permissions
- Review project-related documents
- Generate advanced reports
- Analyze project information


---

## 3. Administrator

### Description
Users responsible for managing the platform.

### Permissions
- Manage users
- Manage roles
- Control system settings
- Monitor platform activity

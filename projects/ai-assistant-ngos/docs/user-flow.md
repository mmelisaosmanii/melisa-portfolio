# User Flow - AI Assistant for NGOs

## 1. User Registration Flow

1. User opens the application.
2. User selects the Register option.
3. User enters personal information:
   - Full name
   - Email
   - Password
   - Organization name
   - Role
4. System validates the information.
5. Account is created.
6. User is redirected to the dashboard.

---

## 2. User Login Flow

1. User opens the Login page.
2. User enters email and password.
3. System verifies credentials.
4. User receives authentication token.
5. User is redirected to the dashboard.

---

## 3. Document Analysis Flow

1. User opens the Documents section.
2. User uploads a document.
3. Backend stores the document information.
4. AI service analyzes the document.
5. AI generates a summary or requested output.
6. User views the result.

---

## 4. AI Chatbot Flow

1. User opens AI Assistant.
2. User writes a question or request.
3. Frontend sends the message to the backend.
4. Backend communicates with the AI service.
5. AI generates a response.
6. User receives the answer.

---

## 5. Report Generation Flow

1. User selects report generation.
2. User provides required information.
3. AI processes the information.
4. System creates a structured report.
5. User can view or download the report.
